# Land of Legacy — Team Performance Suite

## Files
```
public/index.html          ← The app
netlify/functions/api.mjs  ← Database API
netlify/functions/messenger.mjs ← Facebook Messenger webhook
netlify.toml               ← Netlify config
package.json               ← Dependencies
```

## Deploy
1. Push ALL files to GitHub keeping folder structure
2. Netlify → Import from Git → Deploy
3. Enable Neon database extension
4. Redeploy
